# 项目简介

轻量级文档阅读器是一个简洁高效的文档展示工具，专为技术文档和项目说明设计。本项目采用纯前端实现，无需复杂的后端支持，即可快速搭建专属的文档网站。

## 核心特性

文档阅读器提供了一系列实用的功能，满足日常文档展示的各种需求。首先，系统支持多种文档格式的自动识别和处理，包括纯文本文件（TXT）和 Markdown 文档，能够自动将 Markdown 渲染为美观的 HTML 页面。文档目录采用树形结构展示，支持无限层级嵌套，让您可以轻松组织和管理大量文档内容。

搜索功能是本项目的另一大亮点。当在配置文件中启用搜索功能后，用户可以通过侧边栏的搜索框快速定位目标文档。搜索支持文档标题、文件名的模糊匹配，并提供即时搜索结果展示。这一功能在文档数量较多时尤为实用，大大提升了用户的查阅效率。

侧边栏的智能折叠机制能够自动根据文件夹内的文档数量调整展开或收起状态。对于包含 4 个及以上文档的文件夹，系统默认将其收起，帮助用户保持界面整洁；而文档较少的文件夹则默认展开，方便快速访问。同时，用户可以随时点击文件夹名称来展开或收起内容，满足个性化的浏览需求。

暗色模式是现代化文档阅读器不可或缺的功能。本系统提供了完整的暗色主题支持，包括侧边栏、内容区域、代码块、表格等所有元素的颜色适配。主题设置会自动保存到本地存储，下次访问时自动恢复用户上次选择的主题，为长时间阅读提供舒适的视觉体验。

Mermaid 图表支持让您的文档更加生动。通过在 Markdown 中使用 Mermaid 语法，您可以轻松插入流程图、甘特图、时序图、类图等各种图表，满足技术文档的展示需求。系统已内置 Mermaid 库，支持开箱即用。

文档首页功能允许您指定一个文档作为网站首页。当用户首次访问或直接访问网站根路径时，系统会自动加载指定的文档内容，无需手动选择。这一功能特别适合将欢迎文档、使用指南或项目概述设置为首页的场景。

## 技术架构

本项目采用简洁的技术栈实现，前端使用原生 JavaScript 开发，不依赖任何大型框架，确保了代码的轻量性和可维护性。样式采用原生 CSS 编写，支持响应式布局，适配桌面端和移动端设备。所有配置通过 JSON 文件管理，无需编程知识即可完成基本配置。

项目支持一键部署到 GitHub Pages、Netlify、Vercel 等静态托管平台，也可以通过任何静态文件服务器运行。自动同步脚本可以将 txt 目录下的文档自动转换为网页格式，并生成文档索引，简化了文档更新的流程。

## 适用场景

本项目适用于多种文档展示场景，包括但不限于项目技术文档、团队知识库、产品说明书、API 文档、教程指南等。简洁的设计和丰富的功能使其成为个人项目文档和团队内部文档的理想选择。
