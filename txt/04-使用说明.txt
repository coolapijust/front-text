# 使用说明

## 基本操作

### 浏览文档

1. 点击侧边栏中的文档标题即可打开文档
2. 文档会在主内容区域显示
3. 支持多个文档快速切换

### 搜索文档

1. 点击侧边栏上方的搜索框
2. 输入关键词实时搜索
3. 搜索结果高亮显示

### 切换主题

1. 点击侧边栏底部的太阳/月亮图标
2. 主题偏好会自动保存到本地
3. 下次访问自动应用

### 展开/收起侧边栏

1. 点击侧边栏底部的折叠按钮
2. 侧边栏收起后按钮会移到左侧
3. 再次点击可展开

## 文档格式

### 支持的格式

| 格式 | 说明 | 文件扩展名 |
|------|------|-----------|
| TXT | 纯文本，支持 Markdown 语法 | .txt |
| Markdown | 标记语言 | .md |
| Word | Word 文档 | .docx |

### Markdown 语法

#### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

#### 列表

```markdown
- 无序列表项
- 无序列表项

1. 有序列表项
2. 有序列表项
```

#### 代码块

```python
def hello():
    print("Hello, World!")
```

#### 表格

```markdown
| 列1 | 列2 |
|-----|-----|
| 内容1 | 内容2 |
```

#### Mermaid 图表

```mermaid
graph TD
    A[开始] --> B[处理]
    B --> C[结束]
```

## 快捷键

| 快捷键 | 功能 |
|-------|------|
| Ctrl + / | 切换主题 |
| Home | 返回顶部 |
| End | 跳转底部 |

## 常见问题

- **文档不显示**：检查文档是否在 txt 目录下
- **样式异常**：尝试清除浏览器缓存
- **搜索无结果**：确认搜索功能已启用

## 进阶使用

### 设置首页

在 `reader/config.json` 中设置 `home_page` 为指定文档文件名。

### 排除文件

使用 `exclude_patterns` 和 `exclude_files` 排除不需要的目录或文件。
